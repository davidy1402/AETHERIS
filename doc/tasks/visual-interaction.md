# 视觉与交互模块任务

> 模块目标：实现极简苹果风、玻璃拟态、深浅模式、数字滚动、图表动画、背景流光和资产卡片微交互。

## 最小任务

- [x] VIS-001 定义基础视觉主题
  - 在 Tailwind 或全局 CSS 中定义背景、文字、边框、正负盈亏颜色。
  - 验收：深浅主题都有可读对比度。

- [x] VIS-002 实现玻璃拟态卡片样式
  - 为总览卡片、资产卡片、AI 卡片、图表区域统一卡片样式。
  - 使用 `backdrop-blur`、半透明背景、细边框。
  - 验收：卡片风格一致。

- [x] VIS-003 接入深色/浅色模式
  - 安装并配置 `next-themes`。
  - 创建 `ThemeToggle` 并接入 Header。
  - 验收：切换后刷新页面能保持主题。

- [x] VIS-004 实现 `NumberTicker`
  - 使用 Framer Motion 与 transition 过渡实现数字与价格块的动效。
  - 支持价格、盈亏、百分比。
  - 验收：数值变化时不是瞬间跳变。

- [x] VIS-005 接入总览和资产卡片数字滚动
  - 在 `MetricCard` 和 `HoldingsTable` 中加入流畅的变化曲线。
  - 验收：价格轮询后数字有过渡动画。

- [x] VIS-006 实现图表入场动画
  - 折线图（渐变 Area 图）和饼图（Pie Donut）首次出现时有动画。
  - 验收：动画不影响图表读数。

- [x] VIS-007 实现背景流光效果
  - 创建背景的模糊视觉圆环斑点。
  - 验收：背景效果不遮挡内容，不影响点击。

- [x] VIS-008 实现资产卡片 3D 倾斜与交互 (MetricCard 支持弹簧效果)
  - 鼠标悬浮在卡片上时，有流畅的平移与缩放回弹过渡。
  - 验收：卡片 hover 有轻微动能过渡。

- [x] VIS-009 实现资产卡片高光跟随
  - 移入列表或卡片时，文字和边框会呈现微亮态或选择边线。
  - 验收：高光只作为视觉增强，不影响文字可读性。

- [x] VIS-010 做移动端交互降级
  - 移动端禁用或弱化 hover 相关效果，保留布局结构。
  - 验收：触屏设备不会出现异常倾斜或闪烁。

- [x] VIS-011 手动视觉验收
  - 检查桌面、平板、手机三种宽度。
  - 检查深色和浅色模式。
  - 验收：无明显文字重叠、卡片撑破、横向滚动。

## 完成标准

- [x] 本模块所有任务完成。
- [x] 视觉动效不改变业务数据。
- [x] 页面在移动端和桌面端都可读、可用。
